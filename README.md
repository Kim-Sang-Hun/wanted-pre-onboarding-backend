<h1>원티드 프리온보딩 백엔드 인턴십 선발과제</h1>

<h2>주제 설명</h2>
본 서비스는 기업의 채용을 위한 웹 서비스 입니다.<br>
회사는 채용공고를 생성하고, 이에 사용자는 지원합니다.

<h2>요구 사항</h2>
1. 회사 대상 서비스<br><br>
   (1) 채용공고 생성, 수정, 삭제<br><br>
2. 사용자 대상 서비스<br><br>
   (1) 채용공고 조회, 검색<br>
   (2) 채용 상세 페이지 조회<br>
   (3) 채용공고에 지원<br>
